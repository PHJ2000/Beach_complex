# 기술 결정 기록

## 1. 목적
- 이 프로젝트에서 사용하기로 한 주요 기술/라이브러리/아키텍처 결정과 그 이유를 기록합니다.
- 나중에 새로 합류한 사람이 "왜 이걸 쓰고 있지?"라는 질문에, 이 문서를 보고 이해할 수 있도록 합니다.

## 2. 기록 대상
- 프레임워크 선택 (예: Spring Boot, Express 등)
- 데이터베이스 / 캐시 / 메시지 브로커 선택
- 인증/인가 방식 (JWT, 세션, OAuth 등)
- 로깅 / 모니터링 / 배포 방식 등
